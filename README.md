## 4. Functional

- Authenticate and authorization
- Interview question
  - Import question
  - Show question
- Management user
- Responsive on multiple devices such as: laptop, ipad, iphone X...
## 5. How to run

- Build server

```java
$ cd server
$ npm install
$ npm start
```

- Build client

```java
$ cd client
$ npm install
$ npm start
```
